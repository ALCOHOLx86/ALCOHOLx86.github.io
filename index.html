ALCOHOL FOR POSIX

ARBITRARY LAYER for COVERSION with OPERATIONALLY HEAVY OPERATING LAYERS (FOR POSIX)

Alcohol for posix is a set of two projects (espresso and beer) that combine to creat a third (vodka),
the aim of these projects are to enable windows exes on Posix complient systems.

VODKA Operates Dynamic Kernels Arbitrarily (for POSIX)

Vodka is an implementation of Espresso inside of Beer. Espresso is an arbitrary Win to Posix call translation layer
that takes a Win sys call and wrappes it in an arbitrary Posix call in an attempt to load the exe.

BEER POSIXüç∫

Binary Executable Engagable Runtime for POSIX complient systems.

Welcome to BEER POSIXüç∫ , beer is a open sorce brute force binary windows exectutable runtime for intel (x86_64 architecture) computers running a POSIX BSD based Operating System.
BEER loads a exe into memory and then jumps and runs the executable code.

::QUICK DISCLAIMER::
Both espresso and beer are currently very unstable, take into account most exes will crash since this is only a experimental project written in base c.

Download latest release

After downloading the binary, cd into the directory were its saved and run this command to add it to your terminal path,
after this you can use the BEER command to compile and run exe files (very unstable most crash).
The command to add to PATH:

mv BEER /usr/local/bin/
chmod +x /usr/local/bin/BEER
